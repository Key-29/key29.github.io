🎯 认知战策略整理（核心作战体系）

2025-06-17

1️⃣ 反制混淆概念 & 拒绝被代表

🎯 战术目标：打断中共“假装是中国人”的话语霸权

中共常用伎俩：

  • 要打仗时：“中国人民强烈愤怒”
  • 想和恐怖政权结盟时：“我们的老朋友”
  • 制裁来时：“制裁我们中国人”
  • 一旦有好事：“党的正确领导”
  • 一旦出事：“群众情绪复杂”

你的策略：

  • 明确主语：“谁下令、谁代表、谁控制”
  • 提示词方向：补主语、拆句式、反转立场归属
  • 实战效果：让小粉红和五毛无法代入，自我认同断裂

2️⃣ 极限官话生成器（可攻可守）

🎯 战术目标：抢占语言空间，制造信息污染，压制对方气势

用法一：模仿极致官方语言

  • 效果：讽刺、让人反胃、降低接受度
  • 示例：把“人民的热烈期待”用在“要不要打台湾”问题上

用法二：对话压制（反制小粉红）

  • 用标准党八股回复他们，抢夺语境主动权
  • 对方骂你，就用“坚决拥护党中央统一部署”把他怼到无法接话

提示词设计方向：

  • 把一句正常话，生成“六层结构官话”
  • 模拟新华社+外交部联合发言

3️⃣ 严肃问题娱乐化

🎯 战术目标：让中共急迫的意识形态行动变成笑料，失去威慑力

适用对象：

  • 官宣、禁令、辟谣、宣传片、通报批评、微博大V喊话

策略方法：

  • 模仿段子手 / B站弹幕语气 / 老年人朋友圈风格
  • 把整件事转化为“网络笑话模板”

效果：

  • 让“严肃内容”变“低级娱乐”，削弱恐惧感，打断服从链
  • “谁认真谁就输了”，正是让中共最恼火的状态

4️⃣ 拉仇恨策略（反向传播）

🎯 战术目标：把中共本来想“两头骗”的宣传内容，变成“两头都恨它”的结果。

中共长期搞“舆论双轨制”：

  • 在墙内：洗脑、威胁、装代表中国人民
  • 在墙外：装可怜、扮负责任、假装文明

你要做的，就是打通这两条轨道，让他们互相冲撞，引发双边仇恨反应。

🎯 战术一：墙内洗脑内容 → 扔去国际舆论场

目的：让外国人看到中共在墙内干了什么，原话原截图，不加解释。
目标平台：X（前推特）、川普账号、独立记者、国会议员、国际NGO

示例：

  • 把央视吹新疆转运、共青团煽动仇恨的新闻截图 → 扔到川普X下面
  • 把新华社吹嘘“内循环自给自足，老百姓吃不上饭是爱国” → 扔给华盛顿邮报评论员

效果：

  • 外国人看得发疯：“你们真是这么说的吗？”
  • 中共外交部只能两种选项：承认、否认 → 两边都下不来台

🎯 战术二：外宣谎话 → 带回国内传播

目的：把中共对外说的“人话”，贴回它墙内控制不了的角落。
目标平台：知乎、小红书、豆瓣、微信群、战友圈

示例：

  • 把“China is a partner of peace and stability”截图发回国内
  • 把CGTN对外否认镇压新闻的截图 → 发进抗议者讨论组
  • 把中共吹捧非洲援助的宣传片 → 发给正在被裁员的地方公务员

效果：

  • 墙内人看到“原来他在外面这么舔、这么假”
  • 情绪反弹，不用争论，直接恶心到人

🎯 战术三：让中共“外宣”和“内宣”互相打脸

目的：制造“墙内和墙外两种说法冲突”的局面，让它自乱阵脚
方法：对比截图、双语翻译、公开同步发布

示例：

  • 把人民日报中文版本和英文版内容并列 → 发给墙内外都能看到的群组
  • 把外交部英文答记者问，翻译成中文直白版 → 发给微博评论区
  • 把环球时报英文版嘲讽美国自由言论 → 发给国内被禁言网友

效果：

  • 内外认知冲突炸裂
  • 让本来想“当好人”的宣传内容，变成“多方都骂”的灾难

✅ 配套提示词策略方向：

  • 提示词生成结构：

    “请把以下墙内洗脑内容改写成川普账号下的回帖用语，保留中共原文核心表达，语言简洁尖锐。”

    “请生成一条微博或知乎评论，用于贴出中共对外撒谎的外宣内容，让墙内用户直接反感。”

  • 可结合：仇恨引导句式、情绪激化话术、外交部发言反编译结构
