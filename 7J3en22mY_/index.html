🧱 战友内容再加工系统（三种作战模式）

2025-06-18

1️⃣ 群体适配转换（内容转译）

🎯 战术目标：把爆料革命内部的内容，转成不同群体能接受、愿意听、愿意传播的表达方式。

方法关键词：

  • 受众视角切换
  • 情绪接口匹配
  • 立场包装重写

适配对象举例：

  • 小粉红（强调生活焦虑、房贷困局、就业问题）
  • 官迷（制造体制背叛感、揭露高层内斗）
  • 中产/职场人（信息不透明+经济断层）
  • 家庭妇女/长辈（子女安全、医疗教育）
  • 体制内灰色人群（未来不可预期感）

提示词方向：

“请将以下内容改写成适合家庭主妇阅读的风格，强调子女命运、日常生活与政策关联，去掉宏大叙事。”

“把这段爆料内容重写成小红书口吻，重点放在‘生活失控感’和‘假繁荣’上，不提政权和党。”

2️⃣ 政治视角反制（叙事识破）

🎯 战术目标：用政客、战略顾问、媒体操盘手等“高位视角”，识破中共设定的叙事框架并回帖反打。

方法关键词：

  • 框架劫持
  • 立场剥离
  • 角色模拟拆解

常用角色模板：

  • 美国战略顾问
  • 退役国安专家
  • 欧盟外交圈评论员
  • 政治智库研究员
  • 台海问题军事顾问

提示词方向：

“用一个美军智库顾问的角度解读这条央视新闻，指出其中使用的模糊主语、对立构建和情绪操控。”

“模拟政客视角，用一句冷静的话反问这条官媒评论，让墙内五毛哑口无言。”

3️⃣ 深度话语建构（输出高地）

🎯 战术目标：从多角度输出高密度内容，占据舆论高地，抢先设置议题，制造话语权压力。

方法关键词：

  • 倒金字塔结构
  • 多线索并列展开
  • 高频信息密度

适用内容：

  • 财政危机 / 地方债务 / 社保空账
  • 地缘政治 / 制裁博弈 / 对台作战
  • 舆论战术 / AI洗脑 / 宣传结构演化
  • 爆料信源复盘 / 关键事件重建

提示词方向：

“从政治稳定性、经济流动性、军警系统控制力三个角度，写一篇分析中共财政断裂后的政治风险文章。”

“用高密度信息结构，写一篇不带情绪但读完压力巨大的新闻复盘文，语言克制但杀伤力强。”
